README
Project APP: 
Im Projekt App haben wir bisher 7 Klassen und 7 layout erstellt. 
Bisher kann sich über die Klassen RegisterActivity oder MainActivity anmelden oder registrieren. Die Anmeldedaten werden in einer Firebase Database gespeichert.
(für den Zugirff auf die Database bräuchte ich eure Emails, dann kann ich Sie euch geben)
Über die Klasse Startseite kann man zunächst auf 4 verschieden Optionen klicken. Bisher sind nur 2 davon Funktionstüchtig: Eventerstellen und Profil.
Bei Eventerstellen kann man bisher ein Event erstellen und es teilen. Jedoch sind die Daten statisch und nicht funktionstätig
Bei Profil kann man seine eingebenen Daten aus dem Login vorerst ansehen. Auf Einstellungen und ProfilBearbeiten kann man diese verändern und anspassen. 
Mit Bestätigen werden diese Daten automatisch in Database gespeichert und man kann sich seine neu eingebenen Daten anschauen.

Probleme: 
Momentan hänge ich an dem Problem das sich das Profilbild wohl ändern lässt, jedoch wird es nicht mehr angezeigt sobald man die App schliesst.

Ziele: 
Die nächsten Ziele sind das Profil fertigzustellen, die Option der Freunde hinzufügen und Freundeslisten anzuzeigen, bei Eventerstellen ein Algorithmus zu schreiben, 
sodass sich Event erstellen und diese sich per Link teilen lassen. Alle die die App haben kommen über den Link auf App und können bei dem Event teilnehmen. Falls man 
die App nicht installiert hat wird auf eine Website geschickt, bei der man ebenfalls bei dem Event teilnehmen kann. 
Der das Event erstellt hat bekommt eine Übersicht wer und wie viele an dem Event teilnehmen
